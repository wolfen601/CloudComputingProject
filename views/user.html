Our Approach

Given the projects goals, my partner and I decided to start by brainstorming application ideas that would provide an opportunity to cover as many topics as possible. Originally we thought it would be a great idea to try and make a mobile interface to pair with another project we were working on, however we later decided that it would limit the amount of concepts we could cover. We then settled on the idea of creating an application that would calculate a user’s current grade in a course.
